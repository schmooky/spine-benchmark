🎮 Spinefolio Example

This page demonstrates how to use Spinefolio to embed Spine animations in your web pages. All you need is the CSS and JS files in the same directory as this HTML file.

Example 1: Basic Auto-Initialization

Add the data-spinefolio attribute to any element and configure it with data attributes. The widget will automatically initialize when the page loads.

<div data-spinefolio data-skeleton="./low_1.json" data-atlas="./low.atlas" data-animation="idle" data-loop="true" data-show-controls="true" ></div>

Example 2: Custom Background

Customize the background color for better visual presentation.

<div data-spinefolio data-skeleton="./low_1.json" data-atlas="./low.atlas" data-animation="win" data-loop="true" data-background-color="#1a1a2e" data-show-controls="true" ></div>

Example 3: Interactive with Pan & Zoom

Enable pan and zoom controls for interactive exploration of the animation.

<div data-spinefolio data-skeleton="./low_1.json" data-atlas="./low.atlas" data-animation="blur_in" data-loop="true" data-enable-pan="true" data-enable-zoom="true" data-min-zoom="0.5" data-max-zoom="3" data-show-controls="true" ></div>

JavaScript API Usage

You can also create widgets programmatically using the JavaScript API.

// Create a widget programmatically const widget = Spinefolio.create('#programmatic-widget', { skeleton: './low_1.json', atlas: './low.atlas', animation: 'idle', loop: true, showControls: true, enablePan: true, enableZoom: true, onLoad: () => console.log('Widget loaded!'), onError: (error) => console.error('Error:', error) }); // Control the widget widget.setAnimation('win', true); widget.setSkin('default'); widget.setTimeScale(1.5); // Get the widget instance later const instance = Spinefolio.get('#programmatic-widget'); // Destroy when done Spinefolio.destroy('#programmatic-widget');

Available Data Attributes

data-skeleton Path to skeleton JSON file data-atlas Path to atlas file data-animation Animation name to play (default: first animation) data-skin Skin name to use (default: default skin) data-loop Loop animation (default: true) data-scale Scale factor (default: 1) data-x X position offset (default: 0) data-y Y position offset (default: 0) data-background-color Background color (default: transparent) data-fit Fit to container (default: true) data-show-controls Show control panel (default: false) data-allow-skin-change Show skin switch in controls (default: false) data-enable-pan Enable pan/drag (default: true) data-enable-zoom Enable zoom (default: true) data-min-zoom Minimum zoom level (default: 0.25) data-max-zoom Maximum zoom level (default: 4) data-show-bones Show skeleton bones (default: false) data-show-bounds Show bounding boxes (default: false) data-debug Enable debug mode (default: false) data-pma Premultiplied alpha (default: true) data-copyright Copyright text data-font Custom font family
