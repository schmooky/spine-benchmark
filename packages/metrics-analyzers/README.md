# @spine-benchmark/metrics-analyzers

Low-level analyzers used by Spine benchmark metrics.

## Exports

- skeleton analyzer (`analyzeSkeletonStructure`)
- mesh analyzer (`analyzeMeshesForAnimation`, `analyzeGlobalMeshes`)
- clipping analyzer (`analyzeClippingForAnimation`, `analyzeGlobalClipping`)
- blend mode analyzer (`analyzeBlendModesForAnimation`, `analyzeGlobalBlendModes`)
- constraint/physics analyzer (`analyzePhysicsForAnimation`, `analyzeGlobalPhysics`)
- related metric and analysis types
